# Discord бот расписания БР

Простой бот на `discord.py`, который реализует две слэш-команды:

- `/help` — подсказывает, что нужно воспользоваться `/br`.
- `/br` — отправляет embed с расписанием и кнопками выбора типа техники, оформленный как на предоставленном макете. При выборе «Танки» сообщение сразу показывает подборки для БР 12.7–4.7 с актуальным списком техники по странам и позволяет переключаться между значениями; «Авиация» раскрывает подборки для БР 12.7–4.7, повторяя формат предоставленных скриншотов.

## Запуск

1. Создайте приложение в [портале разработчика Discord](https://discord.com/developers/applications) и включите для него слэш-команды.
2. Сохраните токен бота в переменной окружения `DISCORD_TOKEN`.
3. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

4. Запустите бота:

   ```bash
   python bot.py
   ```

После запуска бот автоматически синхронизирует слэш-команды с сервером.
